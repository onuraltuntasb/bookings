## About The Project
<br/>
bookings-backend is a demo project to see how to develop vanilla golang monolith
backend with templates



The three principles of project:

-   Clean templating for future projects 
-   Try to learn templates in golang 
-   Follow golang way (simplestic approach)

## Getting Started

### Prerequisites

-   Go 1.22 or upper

### Installation 

1. Clone the repo
    ```sh
    git clone https://github.com/onuraltuntasb/bookings
    ```
2. Go install dependencies
    ```sh
     go mod tidy
    ```
3. Run bookings
    ```js
     go run .\cmd\web\main.go 
    ```
